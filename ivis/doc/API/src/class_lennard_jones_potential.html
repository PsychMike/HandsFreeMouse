Detailed Description

Simulate the diffusion of molecules (useful for bouncing things around at area). 

 All the hard work done by Maarten Baan: http://www.youtube.com/watch?v=Q6f5ZXyHtfs
 n.b. always users Euler method (didn't bother implemeting Verlet)
EXAMPLE USAGE
 close all; clear all

 x = 1:6;
 y = 1:6;
 v = (rand(6,1)-0.5)*4;
 u = (rand(6,1)-0.5)*4;
 m = ones(6,1);
 b = [0 7 0 7];
 ifi = 1/120;
 myLJP = LennardJonesPotential(x,y,v,u,m,b,.75,ifi);

 for i = 1:1000
     myLJP.calcNextTimestep();
     myLJP.plot();
     pause(ifi);
 end
