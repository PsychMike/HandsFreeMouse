ivisDemo008_playingVideos. Displaying a video file on a PTB screen.

How to use IvVideo to play, pause and close videos. Note that we have now shifted to using function, because otherwise matlab throws errors with regards to calls to singleton getInstance() constructors. If you are running a 64-bit version of matlab don't forget that you will need to install GStreamer (and probably some additional packages)

If you are having problems then Psychtoolbox's MovieDemo.m might be a good place to start

Note that it is vital that the broadcaster package is imported (ivis.broadcaster.*), as this contains a wrapper for PTB's Screen(Flip) which ensures that screen update events are broadcasted throughout ivis.

Requires: ivis toolbox v1.3

Matlab: v2012 onwards
