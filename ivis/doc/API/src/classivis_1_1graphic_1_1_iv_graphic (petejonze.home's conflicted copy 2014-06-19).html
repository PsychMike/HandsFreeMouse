Detailed Description

Screenobject responsible for remembering the position of an image at each point in time. The associated PTB texture can be drawn by invoking the draw() method. The classifier can request the location of the image at any point in time, using the getXY/getX0Y0 methods.

IvGraphic Methods
  • IvGraphic - Constructor.
  • draw - draw texture to screen
  • getXY - query central XY position (at each time, t)
  • getX0Y0 - query top-left XY position (at each time, t)
  • getX - query current (central) X position
  • getY - query current (central) Y position
  • nudge - increment current XY position
  • reset - set (central) XY position and reset the history

IvGraphic Static Methods:

  • loadAll - convert all image files from a given directory into textures
