Detailed Description

Main (static) class for initialising and shutting down ivis. Should be the first and last thing called.

IvMain Methods
  • finishUp - Shut down system and close eye-tracker connection
  • initialise - Prepare the system for launch
  • launch - Open connections and ready outputs
  • pause - Pause data collection
  • validate - Validate IvConfig parameters
See Also
none
Example
IvMain.initialise(./IvConfig_example.xml); [dataInput, params, winhandle, InH] = IvMain.launch();
