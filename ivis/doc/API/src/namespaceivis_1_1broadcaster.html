mlhsInnerSubst< matlabtypesubstitute, varargout > ivis::broadcaster::Screen ( matlabtypesubstitute  varargin)

SCREEN wrapper for PTB's Screen command. 

Intercepts Screen command and inserts a PreFlip broadcast event before
any <tt>Flip</tt> commands

Note, because further objects may be drawn just prior to the final Flip, it is important NOT to use: Screen(DrawingFinished, winhandle) in your script! 

 varargin   whatever PTB's Screen() requires

Returns: 

   varargout   whatever PTB's Screen() returns

Requires: none
